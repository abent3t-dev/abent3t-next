# Logos de Abent 3T

Esta carpeta contiene los archivos de logo para la aplicacion.

## Archivos requeridos

Coloca los siguientes archivos en esta carpeta:

| Archivo | Descripcion | Uso |
|---------|-------------|-----|
| `logo-a3t.png` | Logo principal (colores corporativos) | Login, reportes, documentos |
| `logo-a3t-white.png` | Logo para fondos oscuros (version blanca) | Sidebar, headers oscuros |
| `logo-icon.png` | Solo el icono (cuadrado, 192x192px min) | Favicon, iconos de app |

## Especificaciones

### logo-a3t.png
- Formato: PNG con transparencia
- Dimensiones recomendadas: 360x120px o proporcional
- Uso: Fondos claros (login, documentos PDF)

### logo-a3t-white.png
- Formato: PNG con transparencia
- Dimensiones recomendadas: 360x120px o proporcional
- Colores: Blanco o escala de grises para fondos oscuros
- Uso: Sidebar (fondo #424846), headers oscuros

### logo-icon.png
- Formato: PNG con transparencia
- Dimensiones: Cuadrado, minimo 192x192px (ideal 512x512px)
- Uso: Favicon del navegador, icono de aplicacion

## Colores corporativos

Segun el manual de identidad de Abent 3T:

| Color | Hexadecimal | Uso |
|-------|-------------|-----|
| Verde principal | `#52AF32` | Color corporativo principal |
| Gris oscuro | `#424846` | Color corporativo principal |
| Azul marino | `#222D59` | Fondos y acentos |
| Dorado | `#DFA922` | Acentos y detalles |

## Notas

- La aplicacion tiene fallback a texto estilizado si los archivos no existen
- Los logos deben seguir las guias del manual de identidad corporativa
- Evitar modificar proporciones o colores del logotipo oficial
